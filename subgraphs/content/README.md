# content — Romain Retreat federation subgraph

Logical Federation 2 subgraph for content collections: `Page`, `Post`, `Media`, `Category`, plus redirects/forms/seo/nested-docs surfaces.

- **SDL** — `src/content.graphql` (regenerated by `yarn export:subgraph-sdl`).
- **Entry** — `index.ts` sets `PAYLOAD_LAMBDA_SUBGRAPH=content`.
- **Default port** — `4004`.

## Run

```sh
yarn dev:subgraph:content
# or
docker compose -f docker-compose.federation.yml up content
```
