# groups — Romain Retreat federation subgraph

Logical Federation 2 subgraph that owns the `Group`, `GroupMember`, `GroupPost`, `GroupComment`, `GroupPostLike`, `GroupCommentLike` entities.

- **SDL** — `src/groups.graphql` (regenerated by `yarn export:subgraph-sdl`).
- **Entry** — `index.ts` sets `PAYLOAD_LAMBDA_SUBGRAPH=groups` and starts the shared runtime.
- **Default port** — `4002`.

## Run

```sh
yarn dev:subgraph:groups
# or
docker compose -f docker-compose.federation.yml up groups
```
