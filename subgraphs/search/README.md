# search — Romain Retreat federation subgraph

Logical Federation 2 subgraph for the Payload `@payloadcms/plugin-search` collection (search index + search queries).

- **SDL** — `src/search.graphql` (regenerated by `yarn export:subgraph-sdl`).
- **Entry** — `index.ts` sets `PAYLOAD_LAMBDA_SUBGRAPH=search`.
- **Default port** — `4003`.

## Run

```sh
yarn dev:subgraph:search
# or
docker compose -f docker-compose.federation.yml up search
```
