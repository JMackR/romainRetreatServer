# system — Romain Retreat federation subgraph

Logical Federation 2 subgraph for Payload internals: `payload-folders`, `payload-locked-documents`, `payload-preferences`, `payload-migrations`, `payload-jobs`, `Header`, `Footer`, plus the `_payload*` operations.

- **SDL** — `src/system.graphql` (regenerated by `yarn export:subgraph-sdl`).
- **Entry** — `index.ts` sets `PAYLOAD_LAMBDA_SUBGRAPH=system`.
- **Default port** — `4005`.

## Run

```sh
yarn dev:subgraph:system
# or
docker compose -f docker-compose.federation.yml up system
```
